External CSS

Using External CSS

if you want to apply certain styles to multiple pages you can use an external css page to link a css style sheet to a html page you place the tags
<link rel="stylesheet" href="page_link.css" type="text/css"/>
in the html page

Using Class and ID

you can place a number of related elements into (e.g. the headers of each of these sections) into a set called a class in order to apply styles to all of them at the same time to signify you are apply styles to a class in a css document you prefix the name with a "."

if you have a single unique element you want to place seperate style onto you can give them and id to signify you are applying styles to an id in a css document you prefix the name with a "#"
